# Codewars: Highest and Lowest

## Description
In this assignment, you are given a string of space-separated numbers and must return the highest and lowest number.

### Rules
* The output string must consist of two numbers separated by a single space.
* The **highest** number must come first.
* All numbers are valid `Int32`; no validation is required.
* There will always be at least one number in the input string.

---

## Examples

| Input | Output |
| :--- | :--- |
| `"1 2 3 4 5"` | `"5 1"` |
| `"1 2 -3 4 5"` | `"5 -3"` |
| `"1 9 3 4 -5"` | `"9 -5"` |

---

## Constraints
* Input is a string of numbers separated by spaces.
* Numbers are within the `Int32` range.
